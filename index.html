EVM (Embedded Virtual Machine)

The full name of EVM is Embedded Virtual Machine, which is essentially a general-purpose and streamlined embedded virtual machine. It is composed of a front-end framework for syntax analysis and a back-end bytecode operation. It can run on a single-chip with limited resources.
